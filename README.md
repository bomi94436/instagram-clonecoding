# Instagram clone coding
타입스크립트, 노드, 리액트와 더 친해지고 싶은 마음에 시작하는 클론코딩!

### 개발 환경
- Typescript
- Nodejs
- React, Redux
- MySQL

### 실행 전
1. back/.env_example 파일의 이름을 back/.env 로 바꿉니다
2. .env 파일 안의 변수들을 자신의 환경에 맞게 할당합니다

### 실행
- /back 폴더에서
    ```json
    docker-compose up --build
    ```
    또는
    ```json
    npm start
    ```
- /front 폴더에서
    ```json
    npm start
    ```
